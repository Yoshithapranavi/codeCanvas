Arrays & Manipulation

Master the foundation - 60% of all DSA problems

FOUNDATION
PHASE 1: Foundations Week 1-2

📚 Core Concepts to Learn

  • Array traversal (forward & backward)
  • Array indexing & bounds checking
  • Prefix sum arrays
  • Kadane's algorithm (max subarray)
  • Frequency counting with hashing

🎯 Problems to Solve

Two Sum Max Subarray Buy Stock Move Zeroes Duplicates
🎯 Key Pattern to Recognize: "Find pair/sum/subarray" → Arrays are first choice. Use hash map for frequency/index lookups.
✅ You are ready to move forward when:

✓ Can solve Two Sum in < 15 minutes

✓ Understand Kadane's algorithm completely

✓ Know when to use hash map vs sorting

2️⃣

Binary Search

Transform O(n) → O(log n) for sorted data

FOUNDATION
PHASE 1: Foundations Week 1-2

📚 Core Concepts to Learn

  • Classic binary search template
  • Left/right boundary finding
  • Binary search on answer
  • Identifying monotonic property
  • Rotated array search

🎯 Problems to Solve

Binary Search Rotated Array Sqrt(x) First/Last Answer Search
🎯 Key Pattern to Recognize: "Sorted array + search" → Binary search. Look for monotonic property - can eliminate half search space.
🔍 Recognition Trick: If problem says "efficient search" on sorted data, first thought should be binary search, not linear scan!
✅ You are ready to move forward when:

✓ Can code binary search template from memory

✓ Understand lower bound vs upper bound

✓ Can solve rotated array variant

3️⃣

Two Pointers

Optimize arrays with left & right movements

MEDIUM
PHASE 2: Optimization Week 3-4

📚 Core Concepts to Learn

  • Opposite direction pointers (left, right)
  • Same direction pointers (slow, fast)
  • Array reversal patterns
  • Partition logic
  • Valid palindrome checking

🎯 Problems to Solve

Two Sum II Container 3Sum Remove Dups Palindrome
🎯 Key Pattern to Recognize: "Sorted array + need pairs" → Two pointers. "Compare elements from opposite ends" → Two pointers shrinks problem space.
🔍 Recognition Trick: If problem asks "find two elements" and array is sorted, ALWAYS think two pointers before hash map!
✅ You are ready to move forward when:

✓ Can solve 3Sum in < 20 minutes

✓ Understand when to move left vs right pointer

✓ Know difference between opposite and same direction

4️⃣

Sliding Window

VERY IMPORTANT - Substring/subarray optimal approach

MEDIUM
PHASE 2: Optimization Week 3-4

📚 Core Concepts to Learn

  • Fixed window size approach
  • Variable window size approach
  • Hash map for character frequency
  • Expand & contract pattern
  • Condition checking

🎯 Problems to Solve

Longest Substring Max Consecutive Min Window Permutation Fixed Window
🎯 Key Pattern to Recognize: "Longest/shortest substring/subarray" → Sliding window. Fixed window? Expand from right. Variable? Shrink from left when condition breaks.
🔍 Recognition Trick: See "substring", "subarray", "contiguous" → 90% sliding window! It's that important!
✅ You are ready to move forward when:

✓ Can solve longest substring without repeats < 20 min

✓ Know when to expand vs contract window

✓ Can use hash map to track character count

NOTE: This is a CRITICAL skill. Master it before moving forward!
